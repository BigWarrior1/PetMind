## 数据流向图

### 用户发送消息流程

```mermaid
sequenceDiagram
    participant User as 用户
    participant FE as Vue前端
    participant Go as Go后端
    participant AI as Python AI
    participant LLM as 通义千问

    User->>FE: 输入问题
    FE->>Go: POST /api/v1/messages/stream
    Go->>Go: 保存用户消息到DB
    Go->>AI: POST /api/v1/chat/stream
    AI->>AI: 检索Chroma向量库
    AI->>LLM: 调用qwen-plus
    LLM-->>AI: AI回复
    AI-->>Go: SSE流式响应
    Go-->>FE: SSE流式推送
    FE->>FE: 实时显示AI回复
    Go->>Go: 保存AI回复到DB
```

### 管理员查看会话流程

```mermaid
sequenceDiagram
    participant Admin as 管理员
    participant FE as Vue前端
    participant Go as Go后端
    participant DB as PostgreSQL

    Admin->>FE: 访问 /admin/sessions
    FE->>Go: GET /api/v1/admin/sessions
    Go->>DB: SELECT * FROM sessions
    DB-->>Go: 所有会话列表
    Go-->>FE: 分页数据
    FE-->>Admin: 显示会话列表

    Admin->>FE: 点击查看消息
    FE->>Go: GET /api/v1/admin/sessions/:id/messages
    Go->>DB: SELECT * FROM messages WHERE session_id = ?
    DB-->>Go: 会话消息列表
    Go-->>FE: 消息详情
    FE-->>Admin: 显示消息内容
```
