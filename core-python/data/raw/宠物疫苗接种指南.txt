# 宠物疫苗接种完全指南

## 一、为什么要给宠物接种疫苗

疫苗接种是预防宠物传染病最经济、最有效的手段。通过接种疫苗，可以帮助宠物建立免疫屏障，预防多种致命性传染病，显著降低发病率和死亡率。

### 1.1 疫苗的工作原理

疫苗是将病原体（病毒或细菌）经过灭活、减毒或基因工程处理后制成的生物制品。接种疫苗后：

1. 疫苗中的抗原刺激机体免疫系统
2. 免疫系统识别抗原并产生特异性抗体和记忆细胞
3. 当真正的病原体入侵时，免疫系统能快速识别并清除，防止发病

### 1.2 接种疫苗的好处

- **保护宠物健康**：预防犬瘟热、细小病毒、猫瘟等致命性疾病
- **降低医疗成本**：预防费用远低于治疗费用
- **保护家人健康**：预防狂犬病等人畜共患病
- **社会责任**：降低疾病在宠物群体中的传播

## 二、犬疫苗接种指南

### 2.1 核心疫苗（必须接种）

**核心疫苗**是指针对高致死率、广泛流行疾病的疫苗，所有犬只都应接种。

#### 1. 犬瘟热（Canine Distemper）
- **疾病危害**：高度接触性传染病，致死率达80-90%
- **症状**：发热、呼吸道症状、消化道症状、神经症状
- **保护率**：95%以上

#### 2. 犬细小病毒（Canine Parvovirus）
- **疾病危害**：主要侵害幼犬，引起严重呕吐腹泻，致死率70-80%
- **症状**：剧烈呕吐、血性腹泻、脱水
- **保护率**：95%以上

#### 3. 犬传染性肝炎/腺病毒（Canine Adenovirus）
- **疾病危害**：引起肝炎、呼吸道疾病
- **症状**：发热、腹痛、呕吐、角膜混浊（蓝眼病）
- **保护率**：95%以上

#### 4. 狂犬病（Rabies）
- **疾病危害**：人畜共患病，一旦发病100%致死
- **法律要求**：多数国家和地区强制接种
- **保护率**：99%以上

### 2.2 非核心疫苗（根据风险接种）

**非核心疫苗**是指针对特定地区或特定风险的疾病疫苗，根据犬只的生活环境和风险暴露情况选择性接种。

#### 1. 犬副流感病毒（Canine Parainfluenza）
- **适用对象**：经常去犬舍、宠物店、公园等犬只聚集场所的狗
- **症状**：咳嗽（犬窝咳）、打喷嚏
- **保护率**：70-80%

#### 2. 钩端螺旋体（Leptospirosis）
- **适用对象**：经常户外活动、接触野生动物或积水区域的狗
- **疾病危害**：人畜共患病，引起肝肾衰竭
- **注意事项**：需每年加强，副作用相对较多

#### 3. 犬冠状病毒（Canine Coronavirus）
- **适用对象**：多犬家庭、犬舍
- **症状**：腹泻、呕吐（通常比细小病毒轻）
- **保护率**：60-70%

#### 4. 莱姆病（Lyme Disease）
- **适用对象**：蜱虫高发地区
- **疾病危害**：关节炎、肾病

### 2.3 犬疫苗类型

| 疫苗类型 | 包含疾病 | 适用场景 |
|---------|---------|---------|
| **二联苗** | 犬瘟热 + 细小病毒 | 最基础免疫，适合高风险地区快速保护 |
| **四联苗** | 犬瘟热 + 细小 + 腺病毒 + 副流感 | 幼犬基础免疫 |
| **五联苗** | 四联 + 冠状病毒 | 多犬环境 |
| **六联苗** | 五联 + 钩端螺旋体 | 户外活动多的犬 |
| **八联苗** | 六联 + 其他病原 | 全面保护（部分疫苗不推荐） |

**推荐选择**：
- 幼犬基础免疫：四联或六联
- 狂犬疫苗：单独接种（与多联苗间隔1-2周）

### 2.4 幼犬免疫程序

**标准免疫程序**（美国兽医协会推荐）：

| 年龄 | 疫苗类型 | 说明 |
|------|---------|------|
| **6-8周** | 第一针四联/六联 | 首免，可适当提前至6周 |
| **10-12周** | 第二针四联/六联 | 加强免疫 |
| **14-16周** | 第三针四联/六联 + 狂犬疫苗 | 完成基础免疫 |
| **1岁** | 四联/六联 + 狂犬 | 首次加强 |
| **此后每年** | 四联/六联 + 狂犬 | 年度加强 |

**重要提示**：
- 疫苗间隔时间为3-4周，不要过早或过晚
- 16周前完成3针基础免疫至关重要（母源抗体干扰期）
- 有些兽医可能建议4针免疫（20周龄再加一针）

**特殊情况调整**：
- **高风险地区**（如犬瘟热/细小流行区）：可提前至6周首免，并增加免疫次数
- **母源抗体高**：首免可适当延后至8-10周
- **成年犬初次免疫**：接种2-3针，间隔3-4周

### 2.5 成年犬加强免疫

**传统方案**：每年一次四联/六联 + 狂犬疫苗

**现代方案**（WSAVA推荐）：
- **核心疫苗**：每3年一次（抗体可维持3年以上）
- **狂犬疫苗**：根据当地法律（多数地区要求每1-3年一次）
- **非核心疫苗**：每年一次（如钩端螺旋体）

**抗体检测**：
- 通过抗体滴度检测判断是否需要加强免疫
- 检测费用：100-300元/次
- 若抗体滴度充足，可延后接种

### 2.6 注意事项

**接种前**：
- 确保犬只健康（无发热、腹泻、呕吐等症状）
- 完成驱虫（至少提前3-5天）
- 避免洗澡、运输应激

**接种后**：
- 在医院观察15-30分钟（防过敏反应）
- 3天内避免洗澡、剧烈运动
- 一周内避免更换环境、接触其他犬只
- 观察是否有不良反应（精神沉郁、食欲下降、注射部位肿胀）

**禁忌症**：
- 怀孕母犬（避免使用活疫苗）
- 正在患病的犬
- 正在服用免疫抑制剂的犬
- 近期有过敏史的犬

## 三、猫疫苗接种指南

### 3.1 核心疫苗（必须接种）

#### 1. 猫瘟（猫泛白细胞减少症，Feline Panleukopenia）
- **疾病危害**：高度传染性，致死率80-90%，尤其幼猫
- **症状**：发热、呕吐、腹泻、白细胞极度减少
- **保护率**：95%以上

#### 2. 猫疱疹病毒I型（Feline Herpesvirus，FHV-1）
- **疾病危害**：引起上呼吸道感染，幼猫可致死
- **症状**：打喷嚏、流鼻涕、结膜炎、角膜溃疡
- **保护率**：70-80%（不能完全预防，但可减轻症状）

#### 3. 猫杯状病毒（Feline Calicivirus，FCV）
- **疾病危害**：上呼吸道感染、口腔溃疡
- **症状**：发热、流口水、口腔溃疡、跛行
- **保护率**：70-80%

#### 4. 狂犬病（Rabies）
- **法律要求**：多数地区强制接种
- **疾病危害**：人畜共患病，100%致死

### 3.2 非核心疫苗（根据风险接种）

#### 1. 猫白血病病毒（Feline Leukemia Virus，FeLV）
- **适用对象**：
  - 经常外出的猫
  - 多猫家庭（有FeLV感染猫）
  - FeLV高风险地区
- **建议**：首次接种前先做FeLV抗原检测（确认未感染）
- **保护率**：80%左右

#### 2. 猫传染性腹膜炎（FIP）
- **争议性疫苗**：效果不确切，保护率仅50-70%
- **限制**：仅适用于16周龄以上、冠状病毒抗体阴性的猫
- **建议**：国内较少使用，可根据兽医建议决定

#### 3. 猫衣原体（Chlamydia Felis）
- **适用对象**：猫舍、多猫环境
- **症状**：结膜炎
- **保护率**：60-70%

### 3.3 猫疫苗类型

| 疫苗类型 | 包含疾病 | 适用场景 |
|---------|---------|---------|
| **猫三联（核心）** | 猫瘟 + 疱疹 + 杯状病毒 | 所有猫必须接种 |
| **猫四联** | 三联 + 猫白血病 | 外出猫、多猫家庭 |
| **猫五联** | 四联 + 衣原体 | 猫舍、繁育场 |

**推荐选择**：
- 室内猫：猫三联 + 狂犬
- 外出猫/多猫家庭：猫四联 + 狂犬

### 3.4 幼猫免疫程序

**标准免疫程序**：

| 年龄 | 疫苗类型 | 说明 |
|------|---------|------|
| **8周** | 第一针三联/四联 | 首免 |
| **12周** | 第二针三联/四联 | 加强免疫 |
| **16周** | 第三针三联/四联 + 狂犬疫苗 | 完成基础免疫 |
| **1岁** | 三联/四联 + 狂犬 | 首次加强 |
| **此后每年** | 三联/四联 + 狂犬 | 年度加强 |

**注意事项**：
- 猫比犬更容易受母源抗体干扰，建议至少3针基础免疫
- 三针间隔3-4周
- FeLV疫苗需接种2针（间隔3-4周）

### 3.5 成年猫加强免疫

**传统方案**：每年一次三联/四联 + 狂犬疫苗

**现代方案**（WSAVA推荐）：
- **核心疫苗（三联）**：每3年一次
- **狂犬疫苗**：根据当地法律
- **FeLV疫苗**：每年一次（如仍有风险）

**特殊建议**：
- 室内猫可适当延长核心疫苗接种间隔
- 老年猫（>10岁）可考虑减少疫苗频率，以抗体检测为准

### 3.6 注意事项

**猫疫苗相关肉瘤（VAS）**：
- 极罕见但严重的副作用（发生率1/10,000-1/30,000）
- 注射部位出现肿瘤
- **预防措施**：
  - 选择背部或四肢远端注射（便于手术切除）
  - 避免同一部位反复注射
  - 使用纯化疫苗、佐剂含量低的疫苗
  - 观察注射部位，如肿块持续>3个月或直径>2cm应立即就医

## 四、疫苗品牌选择

### 4.1 进口品牌（推荐）

**犬疫苗**：
1. **美国辉瑞（Zoetis）**：卫佳系列，市场占有率高，效果稳定
2. **美国硕腾（Boehringer Ingelheim）**：瑞倍适系列
3. **荷兰英特威（MSD）**：曾经的金标准品牌
4. **法国维克（Virbac）**

**猫疫苗**：
1. **美国辉瑞（Zoetis）**：妙三多、妙四多
2. **法国维克（Virbac）**：Purevax系列（纯化疫苗，降低VAS风险）
3. **美国硕腾**：Nobivac系列

### 4.2 国产品苗

**优点**：价格便宜（约为进口疫苗的1/3-1/2）

**缺点**：
- 免疫效果不如进口疫苗稳定
- 副作用发生率相对较高
- 冷链运输保障不如进口疫苗

**建议**：
- 经济条件允许优先选择进口疫苗
- 国产疫苗可用于流浪动物救助等大规模免疫

### 4.3 如何识别真假疫苗

**正规疫苗特征**：
- 外包装完整，有防伪标识
- 疫苗瓶有批号、生产日期、有效期
- 注射后医院提供免疫证明，贴有疫苗标签
- 冷藏保存（2-8°C），不能冷冻

**避免假疫苗**：
- 选择正规宠物医院或官方指定接种点
- 不购买来源不明的疫苗自行注射
- 索要免疫证并记录疫苗批号

## 五、疫苗常见问题

### 5.1 为什么打了疫苗还会得病？

**免疫失败原因**：
1. **免疫程序不完整**：未按时接种或漏针
2. **母源抗体干扰**：幼龄动物母源抗体高时影响疫苗效果
3. **个体免疫应答差异**：少数个体对疫苗应答不足
4. **疫苗质量问题**：假疫苗、过期疫苗、冷链断裂
5. **病毒毒株变异**：疫苗株与流行株不匹配
6. **免疫抑制状态**：患病、营养不良、寄生虫感染等

**降低风险**：
- 完成完整的免疫程序
- 选择正规医院和可靠品牌
- 接种前确保健康状态良好
- 必要时进行抗体检测

### 5.2 疫苗有副作用吗？

**常见轻微反应**（10-20%发生率）：
- 注射部位轻微肿胀、疼痛（2-3天消退）
- 轻度精神沉郁、食欲下降（1-2天恢复）
- 轻微发热

**罕见严重反应**（<1%）：
- **过敏反应**：面部肿胀、荨麻疹、呕吐、呼吸困难
  - 多发生在接种后15分钟内
  - 需立即就医，注射肾上腺素、抗组胺药
- **免疫介导性疾病**：极罕见
- **猫疫苗相关肉瘤**（仅猫）：1/10,000-1/30,000

**处理**：
- 接种后在医院观察30分钟
- 发现异常立即就医
- 下次接种时告知兽医，可能需要预防性用药

### 5.3 成年犬猫从未打过疫苗怎么办？

**补种方案**：
- **犬**：接种2-3针核心疫苗（间隔3-4周）+ 狂犬疫苗
- **猫**：接种2-3针核心疫苗（间隔3-4周）+ 狂犬疫苗

**注意**：
- 成年动物首次免疫通常2针即可产生有效免疫
- 为了保险起见，部分兽医建议3针
- 首次免疫后每年或每3年加强一次

### 5.4 怀孕/哺乳期母犬猫能打疫苗吗？

**怀孕期**：
- 避免接种活疫苗（可能致畸或流产）
- 灭活疫苗（如狂犬疫苗）相对安全，但非必要也尽量避免
- **建议**：配种前完成免疫，确保抗体水平充足

**哺乳期**：
- 可以接种灭活疫苗
- 活疫苗需谨慎，咨询兽医

### 5.5 抗体检测有必要吗？

**优点**：
- 科学评估免疫效果
- 避免过度免疫
- 适合老年动物、免疫介导性疾病患者

**缺点**：
- 费用较高（100-300元/次）
- 不是所有疾病都能检测抗体

**适用人群**：
- 疫苗史不明的成年动物
- 担心疫苗副作用的宠主
- 老年宠物（>10岁）

**判断标准**：
- 抗体滴度充足：延后或不接种
- 抗体滴度不足：接种疫苗

### 5.6 可以自己在家给宠物打疫苗吗？

**不推荐原因**：
1. **疫苗保存要求高**：需全程冷链运输，普通渠道难以保证
2. **注射技术**：需要专业知识和操作
3. **过敏反应处理**：严重过敏需立即急救
4. **无免疫证明**：无法办理出国、寄养等手续

**建议**：前往正规宠物医院接种，确保疫苗质量和应急处理。

## 六、疫苗与驱虫的时间安排

### 6.1 推荐顺序

1. **驱虫在前**：接种疫苗前3-5天完成体内驱虫
2. **疫苗接种**：在宠物健康状态良好时接种
3. **间隔期**：驱虫后等待3-5天再接种疫苗

### 6.2 为什么要先驱虫？

寄生虫感染会：
- 削弱免疫系统
- 影响疫苗免疫效果
- 增加疫苗副作用风险

### 6.3 时间安排示例（幼犬）

| 周龄 | 操作 |
|------|------|
| 4周 | 首次驱虫 |
| 6-8周 | 第一针疫苗 |
| 8周 | 第二次驱虫 |
| 10-12周 | 第二针疫苗 |
| 12周 | 第三次驱虫 |
| 14-16周 | 第三针疫苗 + 狂犬疫苗 |

## 七、特殊情况处理

### 7.1 疫苗接种前感冒/腹泻

**推迟接种**：
- 等待完全康复后1-2周再接种
- 接种时确认体温正常、食欲良好

### 7.2 疫苗接种期间接触病犬/猫

**立即就医**：
- 未完成全部免疫的幼年动物抵抗力弱
- 医生可能建议注射高免血清或单克隆抗体提供被动免疫
- 密切观察，出现症状立即治疗

### 7.3 更换疫苗品牌

**可以更换**：
- 不同品牌疫苗的抗原类似，可以互换
- 但建议尽量使用同一品牌完成基础免疫系列

### 7.4 疫苗过期

**严禁使用**：
- 过期疫苗免疫效果无法保证
- 可能增加副作用风险

## 八、总结

疫苗接种是预防宠物传染病的第一道防线，规范的免疫程序可以显著降低疾病风险。宠物主人应：

✅ **按时接种**：严格遵守免疫程序
✅ **选择正规医院**：确保疫苗质量和应急处理能力
✅ **记录免疫信息**：保存好免疫证，记录疫苗品牌、批号、接种时间
✅ **观察反应**：接种后密切观察宠物状态
✅ **综合防护**：疫苗+驱虫+良好饲养管理
